Spring5案例练习

Dec 23, 2020
1 min read
Spring5

  1. 下载最新的Spring5：

    官网下载地址：https://repo.spring.io/release/org/springframework/spring/

    github（Spring5核心组件）：https://github.com/chenjiesong/img_study/tree/main/study

  2. idea创建普通Java工程

  3. 创建文件夹lib，导入核心组件：

    • commons-logging：日志

    • beans：IOC核心部分

    • core：IOC核心部分

    • context：上下文

    • expression：表达式

      ​

    点击文件-> 项目结构 -> 模块 -> 依赖 -> “+” -> 导入JAR -> 选中lib下的jar包 -> 确定

    ​

  4. src创建包，在其创建普通类，输入简单代码

    ​ 

    package spring;
   
public class User {
   
    public void add() {
        System.out.println("test");
    }
}

  5. 创建Spring配置文件，在配置文件配置创建的对象

    src新建xml -> Spring配置文件 -> 使用bean标签配置User对象创建

    ​ 

    <?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">
   
    <!--配置User对象创建-->
    <bean id="user" class="spring.User"></bean>
</beans>

  6. 进行测试代码编写

    新建包，新建普通类，输入测试代码

    ​ 

    package spring.testdemo;
   
import javafx.application.Application;
import org.junit.Test;
import org.springframework.context.ApplicationContext;
import org.springframework.context.support.ClassPathXmlApplicationContext;
import spring.User;
   
   
public class TestSpring {
   
    @Test
    public void testAdd() {
        //1 加载spiring配置文件
        ApplicationContext context = new ClassPathXmlApplicationContext("bean1.xml");
   
        //2 获取配置创建的对象
        User user = context.getBean("user", User.class);
   
        System.out.println(user);
        user.add();
    }
   
}
